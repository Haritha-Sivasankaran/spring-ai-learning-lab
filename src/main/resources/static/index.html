Spring AI Core APIs

Core API

ChatClient API

View Reference Docs

The ChatClient is the core interface for interacting with large language models. It provides a fluent builder API to formulate system prompts, user prompts, bind external tools, and handle model responses synchronously or asynchronously (via Project Reactor Flux streams).

// Build and use ChatClient in a Spring @Service @Service public class StudyBuddyService { private final ChatClient chatClient; public StudyBuddyService(ChatClient.Builder builder) { this.chatClient = builder .defaultSystem("You are a helpful software tutor.") .build(); } public String askQuestion(String question) { return this.chatClient.prompt() .user(question) .call() .content(); // returns LLM response string } }
